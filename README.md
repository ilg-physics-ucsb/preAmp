# Speed of Sound PreAmp

This repository is intended to hold all files related tp the speed of sound preamplifier as well as instructions needed to build this device. This pre amplifier is intended for the microphones used in the speed of sound experiment, and take in a 1/4in jack and give out a BNC output with roughly 100x amplifacation factor. The preamp requires a center positive barrel jack power supply with output between 5VDC and 24VDC.
